## Goal
Add bounded project-run retry logic, explicit stop conditions, and full-project review bundles for autonomous app builds.

## Acceptance Criteria
- [x] Add explicit stop conditions for blocked dependency graphs, repeated failures, validation failure, and operator-configured run limits.
- [x] Implement bounded retry behavior that is observable and does not loop indefinitely.
- [x] Generate review bundles or handoff summaries that distinguish completed, blocked, and untouched tickets.
- [x] Define explicit commit or no-commit behavior for full-project runs consistent with the write-policy and review defaults.
