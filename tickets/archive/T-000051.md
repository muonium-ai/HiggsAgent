## Goal
Implement concrete local run, artifact, and attempt-summary persistence under the documented .higgs/local paths so developers can verify execution and analytics outputs directly.

## Acceptance Criteria
- [x] Dispatcher runs persist concrete local event streams under `.higgs/local/runs/<run_id>/<attempt_id>/events.ndjson`.
- [x] Dispatcher runs persist concrete local artifacts under `.higgs/local/runs/<run_id>/<attempt_id>/artifacts/` and normalized attempt summaries under `.higgs/local/analytics/attempt-summaries.ndjson`.
- [x] Persisted telemetry is discoverable from the first-party run command output.
- [x] Runtime and dispatcher tests cover the persistence path and the validation and write-gate events recorded in the event stream.
