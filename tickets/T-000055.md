## Goal
Implement a first-party HiggsAgent runtime that can call OpenRouter, edit files, infer changed paths, and drive one ticket to review-ready state.

## Acceptance Criteria
- [ ] Add a first-party HiggsAgent command for autonomous single-ticket execution against a repository root and ticket board.
- [ ] The runtime can call OpenRouter, read and edit repository files, and derive the actual changed-file set from observed workspace mutations.
- [ ] The runtime can run configured validation commands, capture validation evidence, and feed that evidence into the existing write gate.
- [ ] Successful runs can claim a ticket, record progress, and move it to `needs_review` without direct frontmatter mutation.

## Notes
- This is the implementation ticket that makes the test-repo expectation possible for one ticket at a time.
